# 커뮤니티 댓글 작성

> ⬆ 상위: [API 명세서](README.md)

### Response Body
```json
{
  "success": true,
  "data": {
    "id": "00000000-0000-0000-0000-000000000000"
  },
  "error": null
}
```
### DTO
- Request: CommunityCommentCreateRequest
- Response: CommunityCommentIdResponse
### Path Parameter
- `postId`: uuid, required.
### Rule
댓글 또는 1단계 대댓글을 작성한다. 대댓글의 대댓글은 허용하지 않는다.
