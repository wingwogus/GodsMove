# 영농일지 생성

API Path: /api/v1/farming-records
Index: RECORD-002
Method: POST
Request: FarmingRecordCreateRequest
Response: FarmingRecordResponse

## Rule

`workedAt`, `farmId`, `cropId`, `workType`이 필수이다. 저장 트랜잭션 안에서
해당 원본 revision의 기록 피드백 `PENDING`을 함께 생성한다.

커밋 후 실제 회원·밭·작물·작업 상세·사진 수·날씨를 조립한 입력 스냅샷을
준비한다. 이 응답에는 생성된 코칭 문구를 포함하지 않는다. 2단계에서는 상태
조회 API에서 준비 상태만 확인하며, LLM 문구 생성은 다음 단계 책임이다.
