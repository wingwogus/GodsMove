# 커뮤니티 댓글 삭제

> ⬆ 상위: [API 명세서](README.md)

### Response Body
```json
{
  "success": true,
  "data": null,
  "error": null
}
```
### Path Parameter
- `commentId`: uuid, required.
### Rule
작성자만 삭제 가능하다. 실제 row는 삭제하지 않고 `isDeleted = true`로 처리한다. 댓글 수정 API는 제공하지 않는다.
