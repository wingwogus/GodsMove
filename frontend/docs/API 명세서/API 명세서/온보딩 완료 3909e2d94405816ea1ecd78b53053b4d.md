## Rule

온보딩 마지막 단계에서 필수 프로필을 한 번에 저장한다. 요청에는 Authorization bearer token이 필요하며, 백엔드는 token의 member id를 사용한다.

성공 시 토큰은 재발급하지 않고 `OnboardingCompleteResponse`를 반환한다. 모든 필수 필드가 저장되면 `onboarding.status = COMPLETE`, `missingFields = []`가 된다.

MVP에서는 약관 동의, 기본 농장 생성, 중간 임시저장을 처리하지 않는다.
