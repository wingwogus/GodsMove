# 커뮤니티 게시글 작성

> ⬆ 상위: [API 명세서](README.md)

### Response Body
```json
{
  "success": true,
  "data": {
    "id": "00000000-0000-0000-0000-000000000000"
  },
  "error": null
}
```
### DTO
- Request: CommunityPostCreateRequest
- Response: CommunityPostIdResponse
### Rule
게시글은 작물 하나에 반드시 속한다. 영농일지를 공유하면 작성자 본인 기록이어야 하고, 기록의 crop이 게시글 `cropId`와 같아야 한다. 커뮤니티 이미지는 최대 5장이다.
