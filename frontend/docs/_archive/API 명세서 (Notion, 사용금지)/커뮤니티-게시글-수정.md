# 커뮤니티 게시글 수정

> ⬆ 상위: [API 명세서](README.md)

### Response Body
```json
{
  "success": true,
  "data": {
    "id": "00000000-0000-0000-0000-000000000000"
  },
  "error": null
}
```
### DTO
- Request: CommunityPostUpdateRequest
- Response: CommunityPostIdResponse
### Path Parameter
- `postId`: uuid, required.
### Rule
작성자만 수정 가능하다. 작물, Q&A 여부, 제목, 본문, 영농일지, 이미지를 모두 수정할 수 있다. 영농일지를 공유하면 작성자 본인 기록이어야 하고, 기록의 crop이 게시글 `cropId`와 같아야 한다.
`mediaIds`는 수정 후 최종 이미지 목록이다. 기존 이미지를 유지하려면 해당 media ID를 포함해야 하고, 목록에서 빠진 기존 이미지는 게시글에서 제거한 뒤 `uploaded_media.status = DELETED`로 처리한다. 새로 추가하는 이미지는 작성자 본인 소유의 `COMMUNITY_POST` 용도 `TEMP` 미디어여야 한다. 이미지 순서는 `mediaIds` 배열 순서를 따르며 최대 5장까지 허용한다. Cloudinary 원본 삭제는 이 API에서 수행하지 않는다.
