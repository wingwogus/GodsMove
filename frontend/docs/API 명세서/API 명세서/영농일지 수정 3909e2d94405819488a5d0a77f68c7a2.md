# 영농일지 수정

API Path: /api/v1/farming-records/{recordId}
Index: RECORD-004
Method: PATCH
Request: FarmingRecordUpdateRequest
Response: FarmingRecordResponse

## Rule

수정하면 기록 `sourceRevision`이 증가한다. 이전 `PENDING` 또는 `READY`
피드백은 `STALE`로 보존하고, 새 revision의 `PENDING`을 같은 트랜잭션에
생성한다.

사진·메모를 포함한 모든 수정은 새 입력 스냅샷 대상이다. 응답에는 코칭 문구를
포함하지 않으며, 준비 상태는 피드백 상태 조회 API에서 확인한다.
