# TokenResponse

API 분류: API Response
태그: [인증] 계정

## Body

```json
{
  "accessToken": "access-token",
  "refreshToken": "refresh-token"
}
```

## Field

- `accessToken`: string, required.
- `refreshToken`: string, required.

로컬 이메일/비밀번호 로그인과 토큰 재발급에서 사용한다.
