# 영농일지 코칭 피드백 재생성

API Path: /api/v1/farming-records/{recordId}/coaching-feedback/regenerate
Index: RECORD-009
Method: POST
Request: None
Response: CoachingFeedbackResponse

## Rule

현재 revision의 `FAILED` 피드백만 같은 행을 `PENDING`으로 되돌려 입력 스냅샷
준비를 다시 요청한다. 현재 revision 행이 없고 과거 `STALE` 행만 있으면 새
현재 revision `PENDING`을 만든다.

현재 `PENDING`, `READY`, `STALE` 피드백의 재생성은 `COACHING_002` (409)로
거절한다. 영농일지는 정상 유지되며, 이 단계는 LLM 코칭 문구를 생성하지 않는다.
