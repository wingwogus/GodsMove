# 회원 공개 프로필 조회

> ⬆ 상위: [API 명세서](README.md)

### Response Body
```json
{
  "success": true,
  "data": {
    "memberId": "00000000-0000-0000-0000-000000000000",
    "nickname": "황기농부",
    "experienceLevel": 2,
    "managementType": "AGRICULTURAL_MANAGER_INDIVIDUAL",
    "profileImageUrl": "https://res.cloudinary.com/example/image/upload/profile.jpg",
    "farms": [
      {
        "farmId": "11111111-1111-1111-1111-111111111111",
        "farmName": "횡성 황기밭",
        "displayRegion": "강원특별자치도 횡성군"
      }
    ],
    "crops": [
      {
        "cropId": "22222222-2222-2222-2222-222222222222",
        "cropName": "황기"
      }
    ]
  },
  "error": null
}
```
### DTO
- Request: None
- Response: PublicMemberProfileResponse
### Header
- `Authorization`: Bearer access token, required.
### Path Parameter
- `memberId`: uuid, required. 조회 대상 member ID.
### Rule
다른 회원의 공개 프로필을 조회한다. 공개 프로필이므로 `email`, `name`, `phone`, `birthDate`는 반환하지 않는다. 농장 주소는 전체 주소를 노출하지 않고 `displayRegion`만 반환한다. `displayRegion`은 주소를 공백 기준 최대 2토큰까지만 사용한다. 작물 목록은 `cropId` 기준으로 중복 제거한다.
