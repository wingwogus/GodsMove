# 작물 이용부위 카테고리 목록 조회

> ⬆ 상위: [API 명세서](README.md)

### Response Body
```json
{
  "success": true,
  "data": [
    {
      "code": "WHOLE_HERB",
      "label": "전초"
    },
    {
      "code": "ROOT_BARK",
      "label": "뿌리/껍질"
    },
    {
      "code": "FRUIT",
      "label": "열매"
    }
  ],
  "error": null
}
```
### DTO
- Response: CropCategoryListResponse
- Item: CropCategoryResponse
### Rule
작물 이용부위 카테고리 목록을 조회한다.
### Category Codes
- `WHOLE_HERB`
- `ROOT_BARK`
- `RHIZOME`
- `LEAF`
- `FLOWER`
- `FRUIT`
- `SEED`
- `STEM_BRANCH`
- `UNKNOWN`
