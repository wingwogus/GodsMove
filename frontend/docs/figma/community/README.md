## Capture Flow

1. Select a community frame/component in Figma.
2. Tell Codex: `캡쳐: <screen or component name>`.
3. Codex reads the current Figma selection and saves a capture document here.
4. After all community designs are captured, say `커뮤니티 디자인 수집 끝`.
5. Codex implements screens from the saved captures, then builds and checks the result.
