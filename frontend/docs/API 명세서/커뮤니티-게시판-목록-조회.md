# 커뮤니티 게시판 목록 조회

> ⬆ 상위: [API 명세서](README.md)

### Response Body
```json
{
  "success": true,
  "data": [
    {
      "cropId": "00000000-0000-0000-0000-000000000000",
      "cropName": "인삼"
    }
  ],
  "error": null
}
```
### DTO
- Response: CommunityBoardListResponse
### Rule
로그인 회원의 `member_crop` 기준으로 커뮤니티에 노출할 작물 게시판 목록을 반환한다. 같은 작물이 여러 농장에 있으면 응답에서 중복 제거한다. `전체` 탭은 프론트 표시용이다.
