# 로그아웃

API Path: /api/v1/auth/logout
Index: AUTH-005
Method: POST
Request: None
Response: Empty success response

## Rule

인증된 member의 refresh token을 폐기하고 `refreshToken` cookie를 만료시킨다. access token 인증이 필요하다.

## Request

None

`Authorization: Bearer <accessToken>` header required.

## Response

성공 시 `success: true`, `data: null`을 반환한다.

응답 헤더에 만료된 `refreshToken` cookie를 `Set-Cookie`로 내려준다.
