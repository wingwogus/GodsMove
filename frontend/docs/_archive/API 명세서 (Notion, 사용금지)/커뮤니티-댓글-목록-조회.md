# 커뮤니티 댓글 목록 조회

> ⬆ 상위: [API 명세서](README.md)

### Response Body
```json
{
  "success": true,
  "data": [],
  "error": null
}
```
### DTO
- Response: CommunityCommentListResponse
### Path Parameter
- `postId`: uuid, required.
### Rule
최상위 댓글과 1단계 대댓글만 반환한다. 삭제된 댓글은 포함하되 `body`를 `삭제된 댓글입니다.`로 반환한다.
