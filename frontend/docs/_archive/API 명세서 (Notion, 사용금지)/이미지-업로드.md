# 이미지 업로드

> ⬆ 상위: [API 명세서](README.md)

### Response Body
```json
{
  "success": true,
  "data": {
    "mediaId": "00000000-0000-0000-0000-000000000000",
    "imageUrl": "https://res.cloudinary.com/example/image/upload/...",
    "status": "TEMP"
  },
  "error": null
}
```
### DTO
- Request: MediaImageUploadRequest
- Response: MediaImageUploadResponse
### Rule
Base64 이미지 데이터를 서버가 Cloudinary에 업로드한다. 업로드 성공 시 `uploaded_media`를 `TEMP` 상태로 생성하고, 온보딩 또는 커뮤니티 게시글 작성/수정에서 `mediaId`로 연결한다.
### Notes
- Authorization bearer token 필요.
- `usageType`은 `PROFILE`, `COMMUNITY_POST`만 허용한다.
- decoded image size 기본 제한은 10MB다.
