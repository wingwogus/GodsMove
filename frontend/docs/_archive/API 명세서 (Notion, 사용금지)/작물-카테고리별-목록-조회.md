# 작물 카테고리별 목록 조회

> ⬆ 상위: [API 명세서](README.md)

### Response Body
```json
{
  "success": true,
  "data": [
    {
      "id": "00000000-0000-0000-0000-000000000000",
      "externalNo": 101,
      "name": "감초",
      "usePartCategory": "ROOT_BARK",
      "usePartCategoryLabel": "뿌리/껍질"
    },
    {
      "id": "11111111-1111-1111-1111-111111111111",
      "externalNo": 103,
      "name": "황기",
      "usePartCategory": "ROOT_BARK",
      "usePartCategoryLabel": "뿌리/껍질"
    }
  ],
  "error": null
}
```
### DTO
- Response: CropListResponse
- Item: CropResponse
### Path Parameter
- `category`: CropUsePartCategory enum code. 예: `ROOT_BARK`.
### Rule
path variable로 받은 이용부위 카테고리에 속한 작물 목록을 조회한다.
### Notes
- 카테고리 코드가 enum에 없으면 요청 바인딩 단계에서 실패한다.
- lifecycle/plantingYear 규칙은 사용하지 않는다.
