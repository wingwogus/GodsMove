# 커뮤니티 게시글 상세 조회

> ⬆ 상위: [API 명세서](README.md)

### Response Body
```json
{
  "success": true,
  "data": {
    "id": "00000000-0000-0000-0000-000000000000",
    "postType": "QUESTION",
    "title": "황기 발아율이 너무 낮아요 원인이 뭘까요?",
    "body": "올해 황기를 처음 시작한 초보입니다...",
    "crop": { "id": "00000000-0000-0000-0000-000000000000", "name": "황기" },
    "author": { "memberId": "00000000-0000-0000-0000-000000000000", "nickname": "황기농부", "profileImageUrl": null },
    "farmingRecordId": null,
    "images": [],
    "commentCount": 3,
    "likeCount": 8,
    "likedByMe": false,
    "createdAt": "2026-07-06T10:00:00"
  },
  "error": null
}
```
### DTO
- Response: CommunityPostResponse
### Path Parameter
- `postId`: uuid, required.
### Rule
삭제된 게시글은 반환하지 않는다. 댓글 수와 좋아요 수는 조회 시 계산한다.
