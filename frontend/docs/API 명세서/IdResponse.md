# IdResponse

> ⬆ 상위: [API 명세서](README.md)

### 용도
생성/상태 변경 API에서 리소스 ID만 반환할 때 사용.
### Fields
- id: uuid, required. 생성 또는 변경된 리소스 ID.
- status: string, optional. 변경 후 상태.
