# 작물 전체 목록 조회

> ⬆ 상위: [API 명세서](README.md)

### Response Body
```json
{
  "success": true,
  "data": [
    {
      "id": "00000000-0000-0000-0000-000000000000",
      "externalNo": 101,
      "name": "감초",
      "usePartCategory": "ROOT_BARK",
      "usePartCategoryLabel": "뿌리/껍질"
    },
    {
      "id": "11111111-1111-1111-1111-111111111111",
      "externalNo": 102,
      "name": "구기자",
      "usePartCategory": "FRUIT",
      "usePartCategoryLabel": "열매"
    }
  ],
  "error": null
}
```
### DTO
- Response: CropListResponse
- Item: CropResponse
### Rule
산림청 약용작물 기본 데이터를 전체 조회한다.
### Notes
- pagination은 사용하지 않는다.
- `lifecycleType`, `defaultUnit`은 사용하지 않는다.
- 작물 분류는 `usePartCategory` 기준이다.
